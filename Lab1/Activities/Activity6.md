# Activity 6: Error messages

1) Naming your file incorrectly, then compiling.
    - ERROR: class ActivitySix is public, should be declared in a file named ActivitySix.java
2) Forgetting a keyword such as void or class
    - Forgetting -> void
    - ERROR: invalid method declaration; return type required
3) Forgetting a quotation mark "
    - ERROR: unclosed string literal
4) Forgetting a parenthesis ( or )
    - ERROR: ')' or ',' expected
5) Forgetting a dot .
    - ERROR: cannot find symbol
      Systemout.print("Hello World!");
6) Using too many or too few braces { or }
    - Add extra "}" at end
    - ERROR: class, interface, enum, or record expected
